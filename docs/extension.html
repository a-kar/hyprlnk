Developer Installation

Extension Not Yet Published

The HyprLnk extension is currently in development and not yet available on the Chrome Web Store or Firefox Add-ons. You can install it manually using developer mode.

Chrome/Edge Installation Steps:

  1. 1

    Download the Extension

    Clone or download the HyprLnk repository from GitHub

    git clone https://github.com/a-kar/hyprlnk.git
  2. 2

    Open Extension Management

    Navigate to chrome://extensions/ (or edge://extensions/ for Edge)

  3. 3

    Enable Developer Mode

    Toggle the "Developer mode" switch in the top right corner

  4. 4

    Load the Extension

    Click "Load unpacked" and select the frontend/extension folder

  5. 5

    Configure HyprLnk Server

    Make sure HyprLnk is running on http://localhost:4381

Firefox Installation Steps:

Manifest V2 Required

Firefox requires Manifest V2. You'll need to convert the manifest.json file or wait for the Firefox-specific version.

  1. 1

    Open Firefox Debugging

    Navigate to about:debugging

  2. 2

    Click "This Firefox"

    Select "This Firefox" from the left sidebar

  3. 3

    Load Temporary Add-on

    Click "Load Temporary Add-on" and select the manifest.json file
